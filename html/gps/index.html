Semantic Tags

This page is going to be about semantic tags.

Why we need to understand semantic tags?

Semantic tags tell the computer and user the meaning of the code.

Main Tag

The main tag is used for all main content on the html page.

Nav Tag

The nav tag is used to display a list of links. If you would like to know more check out w3schools and MDN below:
